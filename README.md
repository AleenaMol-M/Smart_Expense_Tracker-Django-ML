Smart Expense Tracker

A Django-based web application to track, analyze, and predict personal expenses.

🚀 Features (Current)
Add, edit, delete expenses
Category-wise tracking (Food, Bills, Travel, etc.)
Expense visualization using charts
Budget tracking with progress bar
Weekly & monthly anomaly detection
Basic ML-based expense prediction
🛠️ Tech Stack
Django (Python)
HTML, CSS, Chart.js
SQLite
Pandas, NumPy, Scikit-learn
⚙️ Run Locally
python manage.py runserver
📌 Status

🚧 Currently in development — new features being added.
